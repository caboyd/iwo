# Typescript Webgl2 Physically Based Renderer

Demo to practice pbr rendering techniques and more

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.

### Prerequisites

installations of node and npm

### Installing

Download repository locally

install node modules

```
npm install
```

Run development environment

```
npm run dev
```

## Deployment

### Github pages
 
Set github pages source as the master branch /docs folder

Output the minimized and compressed project to the docs folder

```
npm run build
```

commit and push to github
